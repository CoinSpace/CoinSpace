Coin Wallet is a non-custodial multicurrency wallet for multiple platforms.

Supported platforms:
- Web
- iOS
- Android
- macOS
- Windows
- Linux
- Tor

Supported coins:
- Bitcoin
- Bitcoin Cash
- Bitcoin SV. Removed at block 722524.
- Ethereum and ERC20 tokens
- Litecoin
- XRP
- Stellar
- EOS
- Dogecoin
- Dash
- Monero
- Binance Smart Chain and BEP20 tokens
